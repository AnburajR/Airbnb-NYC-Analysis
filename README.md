Airbnb NYC Market Analysis

This project analyzes Airbnb listings in NYC to understand pricing trends, demand patterns, and market efficiency using Python, SQL, and Power BI.

Tech Stack
Python (pandas)
SQL (MySQL)
Power BI

Key Insights
Manhattan has the highest pricing, while Brooklyn shows strong demand efficiency
Affordable listings (<$100) attract higher engagement
Brooklyn leads in high-demand budget listings
Some areas show oversupply with high availability
Demand-to-supply ratio highlights most efficient markets
